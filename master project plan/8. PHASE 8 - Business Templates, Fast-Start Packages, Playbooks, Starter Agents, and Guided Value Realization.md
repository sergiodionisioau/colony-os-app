19. 3 critical edge cases
Edge case 1 — Existing workspace conflict

A user applies a template to an existing workspace that already contains:

overlapping modules
conflicting workflow identifiers
different policy pack versions

Required behavior:

block unsafe merge
show exact conflict graph
offer only governed resolution options
do not mutate workspace state before approval and checkpoint
Edge case 2 — Upgrade vs customization collision

A workspace installed from template v1 has:

customized dashboards
forked workflow
pinned module version
changed KPI thresholds

Template v2 introduces:

updated starter workflow
stricter policy pack
new module dependency

Required behavior:

detect safe drift vs blocking drift
show pinned vs upgradeable items
block destructive overwrite
allow only safe migration path or require manual approval
Edge case 3 — Interrupted onboarding after rollback

A template install partially succeeds, onboarding starts, then a downstream workflow validation fails and rollback occurs.

Required behavior:

rollback template state cleanly
invalidate onboarding state tied to removed components
preserve user-facing explanation
allow restart from clean baseline
no orphan checklist or starter-agent records

25. Acceptance test for Phase 8

Ask these exact questions.

Catalog test

Can a user understand exactly what a template is, who it is for, and what it installs before taking action?

Install test

Can a template launch a useful workspace through a previewed, checkpointed, validated lifecycle with no hidden content?

Onboarding test

Can a user complete role-appropriate onboarding, pause/resume safely, and reach measurable first value?

Upgrade test

Can a template upgrade preserve safe customizations, expose unsafe drift, and avoid destructive overwrite?

Rollback test

Can a failed template action restore prior state cleanly without orphan onboarding, workflows, or starter agents?

Security test

Can starter agents, workflows, data bindings, and evidence remain bounded, declared, and correctly scoped?

Outcome test

Can the platform prove whether a template delivered early business value using real metrics rather than marketing claims?

If any answer is no, Phase 8 is not done.
